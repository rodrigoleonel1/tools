Bases de datos SQL

Las Bases de datos SQL, son bases de datos del tipo Relacionales y estructuradas. SQL = "Structured Query Language" ("Lenguaje de Consulta Estructurado")

El lenguaje con el cual se manipulan se conoce como Transact SQL aunque algunas alternativas comerciales utilizan su propia variante con mínimas diferencias sobre el lenguaje estandarizado ANSI SQL. Algunos de los productos más conocidos, son: SQL Server, Oracle SQL, PostgreSQL y MySQL
